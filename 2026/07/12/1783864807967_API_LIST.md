# PureBilibili API 文档

> 本文档列出了 PureBilibili 项目中使用的所有 API 接口。
>
> - **Base URL (主站)**: `https://api.bilibili.com/`
> - **Base URL (直播)**: `https://api.live.bilibili.com/`
> - **Base URL (Passport)**: `https://passport.bilibili.com/`
> - **Base URL (私信)**: `https://api.vc.bilibili.com/`
> - **Base URL (移动端)**: `https://app.bilibili.com/`
> - **Base URL (搜索)**: `https://s.search.bilibili.com/`
> - **Base URL (空降助手)**: `https://bsbsb.top/api`
> - **gRPC 端点**: `https://app.bilibili.com/`
>
> 源文件: [ApiClient.kt](app/src/main/java/com/android/purebilibili/core/network/ApiClient.kt)
